# ily2-sounds

a collection of drum sounds, oneshots, and acapellas to be used on strudel from the [i love you](https://www.glitch.cool/) discord server

to use this inside of strudel, enter the following line in your strudel pattern:
```
samples('github:kyrsive/ily2-sounds')
```
please support the wonderful glitch.cool collective responsible for all these samples by grabbing the full pack through [here](https://discord.gg/2FyWPTWhX6)
